Виробляємо обладнання для серйозних тренувань

OgnevSport — виробник спортивного обладнання для залів, армрестлингу, пауерліфтингу та кросфіту. Ми не посередники — кожен тренажер виготовляємо самостійно на власному виробництві.

Для кожного замовника — індивідуальне виготовлення: ваш колір, ваш дизайн, ваші параметри. Якісні матеріали, порошкове покриття, деталі з нержавіючої сталі.
